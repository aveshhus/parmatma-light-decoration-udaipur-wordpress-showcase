## Technical Stack
- WordPress v6.7+
- PHP 8.2+
- Elementor Pro (or similar builder) + custom CSS for elegant, festive styling and service sections
- JavaScript for smooth interactions and potential sliders
- Plugins: Elementor, contact forms (WPForms or similar), Yoast SEO, image optimization/cache
- Performance: WebP images, lazy loading for high-res event/decor photos
